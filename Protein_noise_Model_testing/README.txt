Codes (placed in the order of running)
=====

1. filternas_merged_data.pl 

   - Removes genes with NA values and removes columns with >80% NA values 
   - Input file : MERGED_DATA_Yeast.txt
   - Output files : ColNAremoved_MERGED_DATA_Yeast.txt, Filtered_MERGED_DATA_Yeast.txt 

2. filter_dup_genes.pl 

   - Removes duplicate genes from the dataset 
   - Input file : Filtered_MERGED_DATA_Yeast.txt 
   - Output file : Filtered_nondup_MERGED_DATA_Yeast.txt 

3. filter_noncorr_params.pl 

   - Removes features that do not show significant correlation with noise 
   - Input files : Filtered_MERGED_DATA_Yeast.txt, Filtered_nondup_MERGED_DATA_Yeast.txt
   - Output files : CorrFiltered_MERGED_DATA_Yeast.txt, CorrFiltered_nondup_MERGED_DATA_Yeast.txt

4. IndparamImp_all_analysis.pl 

   - Quantifies the importance of each molecular feature by calculating fraction of variation explained and predictive R2 
     values for all features individually 
   - Input file : Filtered_MERGED_DATA_Yeast.txt
   - Output file : Results/IndParam_impact_1000_Iterations.txt

5. IndparamImp_all_analysis_nondup.pl 

   - Quantifies the importance of each molecular feature in data without duplicate genes by calculating fraction of variation      explained and predictive R2 values for all features individually 
   - Input file : Filtered_nondup_MERGED_DATA_Yeast.txt
   - Output file : Results/IndParam_impact_1000_Iterations_nondup.txt

6. find_impactful_param_list.pl 

   - Find features that show substantial value for fraction of variation explained and/or for predictive R2 
   - Input files : Results/IndParam_impact_1000_Iterations.txt, Results/IndParam_impact_1000_Iterations_nondup.txt
   - Output files : Results/Impactful_parameter_list_DM.txt, Results/Impactful_parameter_list_DM_nondup.txt

7. filter_nonimpact_params.pl  

   - Filter out features that do not show significant fraction of variation explained and/or for predictive R2 
   - Input files : Filtered_MERGED_DATA_Yeast.txt, Filtered_nondup_MERGED_DATA_Yeast.txt
                   Results/IndParam_impact_1000_Iterations.txt, Results/IndParam_impact_1000_Iterations_nondup.txt
   - Output files : ImpFiltered_MERGED_DATA_Yeast.txt, ImpFiltered_nondup_MERGED_DATA_Yeast.txt

8. paramcomb_GeneDMmodel_analysis
  
   - Fitting models to training and test datasets to find out the best combination of features that best explain the               variation in the data and show highest predictive power 
   - Input files : Filtered_MERGED_DATA_Yeast.txt, CorrFiltered_MERGED_DATA_Yeast.txt, ImpFiltered_MERGED_DATA_Yeast.txt
   - Output files : Results/SubModel_ResultsGeneDM_1000_Iterations.txt, 
                    Results/Formatted_frvarExpl_SubModelGeneDM_1000_Iterations.txt
                    Results/Formatted_predRsq_SubModelGeneDM_1000_Iterations.txt

9. paramcomb_nondup_GeneDMmodel_analysis.pl 

   - Fitting models to training and test datasets without duplicate genes to find out the best combination of features that        best explain the variation in noise data and show highest predictive power 
   - Input files : Filtered_nondup_MERGED_DATA_Yeast.txt, CorrFiltered_nondup_MERGED_DATA_Yeast.txt, 
                   ImpFiltered_nondup_MERGED_DATA_Yeast.txt
   - Output files : Results/SubModel_ResultsGeneDM_1000_Iterations_nondup.txt, 
                    Results/Formatted_frvarExpl_SubModelGeneDM_1000_Iterations_nondup.txt
                    Results/Formatted_predRsq_SubModelGeneDM_1000_Iterations_nondup.txt


Files
=====

1. MERGED_DATA_Yeast.txt

   - Contains compiled data with noise and values of all other features 

2. ColNAremoved_MERGED_DATA_Yeast.txt 
  
   - Genes with NA values for noise are removed 
   - Columns which contain >80% NA values are removed from the dataset

3. Filtered_MERGED_DATA_Yeast.txt 

   - Generated from file from point 2 by removing rows (genes) with NA values for >=90%  features 

4. Filtered_nondup_MERGED_DATA_Yeast.txt 
   
   - Generated from file from point 3 by removing gene duplicates (Ohnologs) 

5. CorrFiltered_MERGED_DATA_Yeast.txt 

   - Filtered data generated by removing features that do now show any correlation with noise values 

6. CorrFiltered_nondup_MERGED_DATA_Yeast.txt 

   - Filtered data generated by removing features that do now show any correlation with noise values and not containing 
     duplicate genes 

7. ImpFiltered_MERGED_DATA_Yeast.txt 

   - Filtered data generated by removing features that do not have significant impact 

8. ImpFiltered_nondup_MERGED_DATA_Yeast.txt 
   
   - Filtered data without containing duplicate genes and generated by removing features that do not have significant impact
